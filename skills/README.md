# External Skills For Hermes

This directory is intended to be scanned by Hermes through:

```yaml
skills:
  external_dirs:
    - /mnt/d/Codex_Project/intellectual_tutor/skills
```

These skills are repo-tracked and should remain source-controlled.

Hermes still writes its own mutable local skills to:

```text
~/.hermes/skills/
```
